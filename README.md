# Store

Personal item storage management platform for tracking belongings, locations, categories, tags, and images.

## Stack

- Next.js 16
- React 19
- Tailwind CSS 4
- shadcn/ui conventions
- Supabase PostgreSQL later
- Alibaba Cloud OSS later
- WeChat Mini Program client later

## Development

```bash
npm install
npm run dev
```

Open http://localhost:3000.
